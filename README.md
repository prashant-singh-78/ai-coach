# 🤖 AI Interview Coach

An AI-powered Interview Preparation Platform designed to help students and professionals improve their interview skills through AI-generated questions, resume analysis, ATS scoring, interview analytics, and personalized feedback.

---

## 🚀 Features

* 🔐 User Authentication (Register/Login)
* 👤 User Dashboard
* 📄 Resume Upload & Management
* 📊 ATS Resume Score Analysis
* 💼 Job Description Matching
* 🤖 AI-Based Interview Question Generation
* 🎤 Technical & HR Interview Practice
* 📈 Interview Performance Analytics
* 📝 Personalized AI Feedback
* 📚 Topic-wise Interview Preparation
* 📱 Responsive UI
* ⚡ FastAPI Backend
* ⚛️ React + Vite Frontend

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/prashant-singh-78/ai-coach.git
cd ai-coach
```

### 2. Backend Setup

```bash
cd AI-Interview-Coach/backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```
